Explorando Animações jQuery

fadeIn() / fadeOut()

Clique no botão para ver o elemento desaparecer e reaparecer com efeito de transparência.

Elemento com Fade

Este elemento usa fadeOut() e fadeIn() para desaparecer e reaparecer.

slideToggle()

Clique no botão para o elemento deslizar para cima e para baixo.

Elemento com Slide

Este elemento usa slideToggle() para deslizar suavemente.

fadeToggle()

Clique no botão para alternar entre visível e invisível com efeito de desvanecer.

Elemento com Fade Toggle

Este elemento usa fadeToggle() para alternar a transparência.

Dica: Abra a consola do navegador (F12) para ver as mensagens de log das animações.
